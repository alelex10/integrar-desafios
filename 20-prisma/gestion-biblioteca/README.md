## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

# Base de datos

La base de datos es de tipo PostgreSQL, se leventa con Docker y se encuentra en el archivo docker-compose.yml

### Docker

```bash
$ docker compose up -d
```

### Prisma Studio

Se puede acceder a la interfaz de Prisma Studio desde la URL `http://localhost:5555`

```bash
$ pnpm prisma studio
```

### Registrar un nuevo préstamo

**bash**

```bash
curl --location 'localhost:3000/loan' \
--header 'Content-Type: application/json' \
--data '{
    "userId": "2763b4b4-e73e-4123-91b8-3f340b66fba5",
    "bookId": "06c2f9d6-06c8-4970-b7aa-691d196af9ec"

}'
```

No se como hacerlo en powershell

### Consultar libros disponibles

**bash**

```bash
curl --location 'localhost:3000/book/available-books'
```

### Mostrar historial de préstamos por usuario

**bash**

```bash
$ curl --location 'localhost:3000/user/history-loans/{userId}'
```
